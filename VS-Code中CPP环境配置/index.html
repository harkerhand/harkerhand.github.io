VS Code中C++环境配置

本文最后更新于 2024年3月2日 晚上

前言

前几日身边有人开始学习C语言, 便又重新拾起了自己的兴趣

但已经许久没有使用VS Code来调试运行C++程序, 本文记录我重新配置环境的过程

Steps

VS Code的下载安装

由于国内下载速度极为缓慢, 甚至网站都无法打开, 我推荐两种方法

  • 使用CDN加速

    参考文章国内下载VS Code速度慢问题解决 - 知乎

    将下载地址.net及之前的部分换成vscode.cdn.azure.cn即可高速下载

  • 使用国内下载站

    腾讯软件中心

    缺点是版本无法自选,但起码能用

安装过程无需多言,按需勾选
