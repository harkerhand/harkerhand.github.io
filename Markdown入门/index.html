什么是Markdown?

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，Markdown文件的后缀名便是“.md”。

工欲善其事，必先利其器！

作者使用Visual Studio Code进行编辑
安装插件Markdown All in One和Markdown Preview Enhanced

注意

本文并非Markdown官方教程，仅介绍个人常用语法
